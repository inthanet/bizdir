    define('DB_HOSTNAME','localhost');

    4. Change the website full path "https://directoryfinder.net/demo/business-directory-template/" as "YOUR LOCAL HOST PATH NAME (htts://localhost/business-directory-template/)"

Oops something wrong issue(Data not insert)

Reason for the issue:

  1. SQL: Set default value to NULL for all columns without default value

Symptoms of the issue:

  1. Oops something wrong - message
  2. All PHP scripts stopped working because they report I haven't defined a default value for some columns.
  3. Forms not working

Solution for the issue:

  1. Disable the MySQL strict mode
  2. Then upload our demo Database
  3. Done

Steps to check MySQL strict mode enbled or not:

  1. Goto phpMyAdmin -> Select your Database -> Click SQL tab, now you getting Run SQL query/queries on database window then copy & paste the below code and run.
  2. ->To check strict mode run the this sql: SHOW VARIABLES LIKE 'sql_mode';
  3. Done
  4. ->If one of the value is STRICT_TRANS_TABLES then strict mode is enabled, else not.

If strict mode is enabled means:

- To Change it permanently, If you have WHM access do the following
  1. Type in command line/Terminal sudo nano /etc/mysql/my.cnf
  2. Then add the following
    [mysqld]
    sql_mode=
  3. Save the file and restart the MySQL server
- If you don't have WHM access do the following
  1. Please contact your hosting/server provider.
  2. Inform them to Disable MySQL strict mode

How to Enable or Disable MySQL strict mode in local server:

  1. Goto phpMyAdmin -> Select your Database -> Click SQL tab, now you getting Run SQL query/queries on database window then copy & paste the below code and run.
  2. ->To Disable strict mode run the this sql: set global sql_mode='';
  3. ->To Enable strict mode run the this sql: set global sql_mode='STRICT_TRANS_TABLES';
  4. Done
